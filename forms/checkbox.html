How To Use

Basic Use HTML
<div class="gov-checkbox-radio-group">
    <label class="gov-checkbox"><input type="checkbox"> <i class="icon"></i> Checkbox 1</label>
</div>

This is the common markup for gov custom checkbox. To use checkbox 2 just change <label class="gov-checkbox"> to <label class="gov-checkbox-2">. For color variation just add the color class like for secondary color <label class="adomx-checkbox secondary"> and for inline checkbox add inline class to <div class="gov-checkbox-radio-group">
